# Security Policy

OrbitCab is a locally-run audio plugin with no network service, so the attack surface is
small. Still, if you find a security issue — for example a crash or memory-safety bug
triggered by a crafted preset / `.orbitcab` file or impulse response, or anything
exploitable — please report it **privately** instead of opening a public issue.

**Contact:** band@darwinscat.com

Please include steps to reproduce, your host and OS, and a sample file if relevant. We'll
acknowledge within a reasonable time and, if you'd like, credit you once a fix ships.
